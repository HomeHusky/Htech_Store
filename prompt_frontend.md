Build a premium, high-conversion E-commerce Storefront for "HTech Store" - a high-end retailer for iPhones, Laptops, and PC Gaming. 
Style: Apple-inspired, minimalist, ultra-clean, using Next.js, Tailwind CSS v4, and Shadcn UI.
Color Palette: Pure white/Black (#000000), Royal Blue (#0071e3) for accents, and subtle glassmorphism.

Key Components to generate in one cohesive layout:
1. Navigation: Sticky glassmorphism header with a slim notification bar (e.g., "Trade-in iPhone 15 Pro, get up to 20M VND").
2. Hero Section: Large, cinematic typography. Use a split layout: Left side has bold value propositions ("Titanium. Strong. Light."), Right side has a 3D-effect product showcase (iPhone 15 Pro).
3. Category Grid: Modern bento-box style grid for iPhone, MacBook, iPad, and Accessories with hover-lift effects.
4. Product Section: High-converting cards with:
   - Subtle "New" or "Hot" badges.
   - Quick-add-to-cart button that appears on hover.
   - Micro-interactions for color swatch selection.
5. AI Sales Concierge: A modern floating chat bubble that expands into a sophisticated AI interface with quick action buttons (Consult PC build, Check Warranty).
6. Trust Signals: Minimalist section for Free Shipping, 24-month Warranty, and AI-powered Support.
7. Footer: Organized, clean links with a newsletter signup that feels premium.

Requirements:
- Mobile-first responsive design.
- Use Lucide-React icons.
- Add subtle Framer Motion-like entrance animations for sections.
- Focus on "Selling": clear CTAs, urgency triggers (stock count), and social proof (ratings).

Build a sophisticated Admin Dashboard for "HTech Store" to manage products, orders, and AI settings.
Style: Modern SaaS, clean sidebar navigation, dark-mode friendly, using Next.js and Shadcn UI.

Key Modules:
1. Analytics Overview: Dynamic charts (Recharts) showing Revenue, Conversion Rate, and Top Selling Categories. Include a "Live Activity" feed of orders.
2. Product Management: A dense but readable table with:
   - Status badges (Active, Out of Stock, Draft).
   - Instant inline edit for Price and Stock.
   - "AI Optimization" button to auto-generate SEO descriptions.
3. Order Management: List with "Payment Proof" verification workflow. Add a "Quick Action" to update status (Pending -> Shipping -> Completed).
4. AI Agent Configuration: Specialized settings page to:
   - Edit System Prompts.
   - Select LLM Models (Gemini, GPT-4, Phi-4).
   - Test AI response in a split-view playground.
5. Customer CRM: List of users with their purchase history and AI chat logs to understand customer intent.
6. Inventory Alert: A "reasonable" management feature: Automated alerts for low stock and "Predictive Restock" based on sales velocity.

Design Tone: Professional, data-driven, using a lot of whitespace and subtle borders. High-density information but never cluttered.
